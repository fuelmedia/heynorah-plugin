# HeyNorah WordPress Plugin

Public distribution repository for the HeyNorah WordPress plugin.

- Runtime plugin files live in `heynorah-v2-plugin/`.
- Installable ZIP: [heynorah-v2-plugin.zip](https://raw.githubusercontent.com/fuelmedia/heynorah-plugin/main/heynorah-v2-plugin.zip)
- Update manifest: [update.json](https://raw.githubusercontent.com/fuelmedia/heynorah-plugin/main/update.json)

Development source is kept in the private `fuelmedia/heynorah-v2-plugin` repository.
